答题会话数据检查工具

使用说明：
  1. 打开浏览器开发者工具（F12 或 右键-检查）
  2. 切换到 Console（控制台）标签
  3. 点击下面的按钮运行相应的检查
  4. 查看控制台输出的结果

手动运行代码

您也可以直接在控制台运行以下代码：

// 查看所有会话的详细信息 const sessions = JSON.parse(localStorage.getItem('answerSessions') || '[]'); console.log('所有会话详情：'); sessions.forEach((session, index) => { console.log(`会话 ${index + 1}:`, { source: session.source, questionsAnswered: session.questionsAnswered, totalQuestions: session.totalQuestions, startTime: session.startTime, endTime: session.endTime }); }); // 查看为什么会话的questionsAnswered是0 const wrongSessions = sessions.filter(s => s.source === 'wrong'); console.log('错题会话：', wrongSessions);
