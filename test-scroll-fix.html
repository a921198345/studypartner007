📜 滚动行为修复测试

暂停后可以自由滚动查看历史消息

🎯 修复的功能
  • 暂停AI回答后，对话框不会自动滚动到底部
  • 用户可以手动向上滚动查看历史消息
  • 当用户滚动离开底部时，显示"回到底部"按钮
  • 发送新消息时自动滚动到底部
  • AI正在回答时，如果用户在底部则自动滚动
🔧 技术实现

1. 添加了自动滚动状态管理：

const [autoScroll, setAutoScroll] = useState(true);

2. 监听用户滚动行为：

const handleScroll = useCallback(() => {
  const isAtBottom = scrollHeight - scrollTop - clientHeight < 50;
  if (!isAtBottom && !isStreaming) {
    setAutoScroll(false);
  }
}, [isStreaming]);

3. 暂停时关闭自动滚动：

const handleStopGeneration = () => {
  // ... 暂停逻辑
  setAutoScroll(false);
};

🧪 测试步骤

  1. 在法考问答页面发送一个会产生长回答的问题
  2. 等待AI开始生成回答
  3. 点击暂停按钮（红色方块图标）
  4. 尝试向上滚动查看历史消息
  5. 验证对话框不会自动跳回底部
  6. 滚动离开底部后，查看是否出现"回到底部"按钮
  7. 点击"回到底部"按钮，验证是否滚动到底部
