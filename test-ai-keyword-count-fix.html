AI关键词题目数量显示修复测试

问题描述

从AI聊天点击"相关真题"跳转到题库页面时，显示的题目数量不正确。例如：

  • 搜索"盗窃"应该显示7道题目，但显示了253道（刑法总数）
  • 切换科目筛选后，题目数量显示错误
  • AI关键词筛选状态在页面刷新后丢失

修复内容

  1. ✓ 修改actualTotalQuestions初始值为0，避免显示默认值
  2. ✓ 添加sessionStorage持久化AI搜索状态
  3. ✓ 在学科切换时保持AI关键词状态
  4. ✓ 添加AI搜索提示栏和清除按钮
  5. ✓ 增加调试信息显示实际题目总数

测试步骤

  1. 测试AI关键词搜索跳转
    期望结果：显示"共 7 道题目（以下是盗窃的相关题目）"
    模拟从AI聊天跳转（盗窃）
  2. 测试切换科目后保持AI筛选
    期望结果：切换到"全部科目"后，仍然显示盗窃相关的题目数量，而不是所有题目

    步骤：

    • 点击上面的链接跳转到题库
    • 将科目切换为"全部科目"
    • 检查题目数量是否仍然正确
  3. 测试页面刷新后状态保持
    期望结果：刷新页面后，AI关键词筛选状态仍然保持

    步骤：

    • 在AI关键词筛选状态下刷新页面
    • 检查是否仍显示AI搜索提示栏
    • 检查题目数量是否正确
  4. 测试清除AI筛选
    期望结果：点击"清除筛选"后，显示当前科目的所有题目

    步骤：

    • 在AI筛选状态下点击"清除筛选"按钮
    • 检查题目数量是否变为科目总数
    • 检查AI搜索提示栏是否消失
  5. 测试手动搜索覆盖AI筛选
    期望结果：手动输入搜索词后，AI筛选被清除

    步骤：

    • 在AI筛选状态下，在搜索框输入其他关键词
    • 检查AI搜索提示栏是否消失
    • 检查显示的是新搜索词的结果
